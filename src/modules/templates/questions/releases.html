Releases

A package on NPM will often have more than one release. There are 25,381 packages (30%) that only have one release (that first published package).

The median value is 3 releases. 52% of packages on NPM have 3 releases or more.

Packages whose major version is greater than 1.* actually have a higher median value of 7, and a mean of 13.

The package with the highest number of releases is yarnlang, with 410 releases! It’s current version number is ‘0.1.0-176’.

The higher a package’s major version is, the more likely it is to have more releases.
