NPM By Dependents

We spent some time tracking all the dependencies on every package in NPM. To do this we followed the dependents tree for each package, counting unique dependents.

What this means is that if package a has one dependent b, but then there are 100 packages that depend on b, that means a actually has 101 dependents.

The majority of packages on npm, 70,828 (75%), have no packages depending on them at all.
